This project focuses on building an AI assistant that helps people write blogs easily and 
quickly. We are using advanced language models called Large Language Models (LLMs), 
which can understand and generate natural language text very well. Blogging is an important 
way to share ideas, stories, and information online, but writing good blogs can take a lot of 
time and effort. Our project aims to create a helpful assistant that generates blog content based 
on user inputs, suggests images, and allows customization in tone and structure. We studied 
many research papers about AI writing tools and found some common issues like unsafe 
content, lack of customization, and poor image support. Our assistant is designed to solve these 
problems by carefully checking content safety, allowing users to set tone and style, and fetching 
related images safely.
